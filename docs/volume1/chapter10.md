# 第10話：「本当の依頼」（優美視点）

「優美、ちょっと話があるんだ」

　実験結果をまとめた論文の初稿が完成した夜、雅史先輩がそう切り出した。

「なんですか？」

「君に正直に聞きたいことがある」

　先輩の表情が、いつもより真剣だった。

「最初に研究に協力してくれた時、本当の理由は何だった？」

　私の心臓がドキッとした。実は、ずっと言えずにいたことがある。

「それは...」

「『プログラミングを学びたい』だけじゃなかったよね？」

　先輩の鋭い指摘に、私は観念した。

「...はい」

　深呼吸してから、本当のことを話し始めた。

「実は、弟がいるんです」

「弟？」

「中学3年生。私とは7歳離れてます」

　私は自分のスマートフォンを取り出して、弟の写真を見せた。

「翔太っていいます。小学校の頃からプログラミングが大好きで」

「へえ」

「Scratchではすごい作品を作るんです。ゲームとか、アニメーションとか」

　私は翔太の作品をいくつか見せた。確かに、小学生とは思えないクオリティ。

「すごいじゃないか」

「でも、中学に入ってPythonの授業が始まったら...」

　私の声が小さくなった。

「最初のエラーで、完全に諦めてしまったんです」


◇◇◇


「先輩の妹さんと同じですね」

「うん...似たような話だ」

「でも翔太の場合、もっと深刻で」

　私は続けた。

「『僕にはプログラミングの才能がなかった』って言って、Scratchも触らなくなったんです」

　あの時の翔太の落ち込みようを思い出すと、今でも胸が痛む。

「あれだけ好きだったのに...『兄ちゃんにはできないことがあるんだ』って」

「それで、君が...」

「はい。私がプログラミングを学んで、翔太を助けたかった」

　これが、本当の動機だった。

「でも、私も同じエラーで挫折して。それで先輩に助けを求めました」

　雅史先輩が優しく微笑んだ。

「なるほど。だから君は必死だったんだ」

「すみません。最初から正直に言うべきでした」

「いや、君のおかげで僕も気づけたことがたくさんある」

　先輩は首を振った。

「でも、翔太君は今どうしてる？」

「相変わらずです。プログラミングの話をすると、すぐに話題を変えてしまいます」

　半年前から変わらない状況だった。

「一度、翔太君と話してみたい」

「え？」

「君の弟なら、きっと僕たちの研究に協力してくれるはず」

　意外な提案だった。

「でも、翔太は人見知りで...」

「大丈夫。僕も昔は人見知りだった」

　先輩が立ち上がった。

「今度の土曜日、君の家に行ってもいい？」


◇◇◇


　土曜日の午後。翔太は最初、部屋から出てこなかった。

「お姉ちゃんの研究の人でしょ？僕には関係ない」

　ドア越しに聞こえる翔太の声。やっぱり人見知りしている。

「翔太、5分だけでいいから」

「嫌だ」

　困った私に、雅史先輩がささやいた。

「ちょっと試してみたいことがある」

　先輩は翔太の部屋のドアの前で、パソコンを開いた。

「翔太君、君のScratch作品、見せてもらったよ」

「...」

「『宇宙戦争ゲーム』、すごく面白かった。特に敵の動きのアルゴリズム」

　ドアの向こうが静かになった。

「あの複雑な軌道計算、どうやって実装したの？」

　しばらく沈黙が続いた後、ドアがゆっくり開いた。

「...本当に見たの？」

「もちろん。ここの部分」

　先輩が画面を見せると、翔太の目が輝いた。

「これ、三角関数使ってるんだ。Scratchでsin波とcos波を組み合わせて」

「すごいな。中学生でそこまで理解してるのか」

　翔太が嬉しそうに説明を始めた。プログラミングの話になると、人見知りが嘘のように饒舌になる。

「でも、Pythonではできないんでしょ？」

　先輩が穏やかに聞いた。

「...うん。エラーが出て、意味が分からなくて」

　翔太の表情が曇った。

「そのエラー、見せてもらえる？」

「でも...」

「大丈夫。怒らないから」


◇◇◇


　翔太が恐る恐るPythonを起動した。

```python
import math
print(math.sin(angle))
```

　予想通り、NameErrorが出た。

「ほら、また出た...」

　翔太が画面から目を逸らした。でも先輩は違った。

「翔太君、これは簡単に解決できるよ」

「え？」

「`angle`が定義されてないだけ。`angle = 90`って一行追加すればいい」

　先輩が実際にコードを修正して見せた。

```python
import math
angle = 90
print(math.sin(angle))
```

　実行すると、きちんと値が表示された。

「え...こんなに簡単？」

「うん。Scratchでも変数は使ってたでしょ？Pythonでも同じ」

　翔太の目がだんだん輝いてきた。

「じゃあ、これは？」

　翔太が別のエラーを見せた。今度はSyntaxError。

「あ、これも簡単」

　先輩は私たちの研究成果を使って、優しく説明した。

「括弧の閉じ忘れ。ここに`)`を追加するだけ」

「そうか...Scratchでもブロックはちゃんと繋げないといけないもんね」

　翔太が理解した瞬間、私は感動した。

　恐怖が好奇心に変わる瞬間を、目の前で見ることができた。

「翔太君、実は君みたいな人のために研究してるんだ」

　先輩が真剣に言った。

「僕？」

「プログラミングが好きなのに、エラーで諦めてしまう人」

　翔太が私を見た。

「お姉ちゃんも？」

「私も同じだった。でも先輩に教えてもらって、恐怖が消えた」

「そっか...」

　翔太がパソコンに向かった。

「もう一回、やってみる」


◇◇◇


　2時間後、翔太は簡単なゲームを作り上げていた。

「できた！」

　画面には、Scratchで作ったゲームをPythonで再現したものが動いている。

「すごいじゃないか」

　先輩が褒めると、翔太が照れくさそうに笑った。

「Pythonって、Scratchより難しいけど、できることが多いんだね」

「そうなんだ。でも基本的な考え方は同じ」

「うん。分かった」

　翔太が私に向かって言った。

「お姉ちゃん、ありがとう」

「え？」

「お姉ちゃんが研究してくれたから、僕またプログラミングできるようになった」

　涙が出そうになった。これが、私の本当の目標だった。

「でも翔太、これからも分からないことあるよ？」

　先輩が言った。

「大丈夫。今度は怖くない」

　翔太が自信を持って答えた。

「エラーは『敵』じゃなくて『ヒント』だって分かったから」

　私は雅史先輩を見た。この人のおかげで、翔太を救うことができた。

　でも、それだけじゃない。

　私たちの研究は、翔太のような子供たちを救うための、本当の第一歩になった。

「先輩」

「ん？」

「これからも、一緒に研究させてください」

「もちろん。君がいないと、この研究は成立しない」

　先輩が微笑んだ。

「翔太君もときどき協力してくれる？新しいアイデアをテストするのに、君みたいな視点が必要だから」

「はい！」

　翔太が元気よく答えた。

　夕日が部屋を照らしている。

　私たちの研究は、まだ始まったばかり。

　でも、確実に誰かを救えることが証明された。

　技術と心理学の融合で、もっと多くの人を救える。

　そんな未来に向かって、私たちは歩き続ける。
